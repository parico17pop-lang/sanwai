
サンワイ・スマイルバンク：静的サイト一式
================================================

このフォルダは、そのままサーバにアップロードすれば動作する静的サイトです。
index.html がトップページです。

構成
----
- index.html（ホーム）
- about.html（私たちについて）
- services.html（事業と取り組み）
- investment.html（投資（為替））
- bamboo.html（竹工房）
- kids.html（子ども食堂支援）
- impact.html（実績・インパクト）
- blog.html（お知らせ）
- company.html（会社概要）
- contact.html（お問い合わせ）
- assets/css/style.css（スタイル）
- assets/js/main.js（JS）
- assets/img（ロゴ・ヒーロー等の画像：ダミー）

使い方
------
1) 画像差し替え：assets/img/ 内のダミーを実写・作品写真に置き換えてください。
2) 文言修正：各ページの本文を実績・方針に合わせて編集してください。
3) デプロイ：
   - さくら/ロリポ/エックスサーバなどにアップロード
   - または GitHub Pages/Cloudflare Pages/Netlify で公開
4) お問い合わせフォーム：現在はサンプル動作（アラート表示）です。
   - 実運用する場合は、送信先（メールサーバ/Google フォーム等）を設定してください。

アクセシビリティ/SEO
---------------------
- スキップリンク、コントラスト、フォーカスリングを配慮
- メタディスクリプション、構造化データ（Organization）を設定済み
- 画像の代替テキスト記載
